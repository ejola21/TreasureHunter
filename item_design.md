# item_design.md — 미션 디자이너용 아이템 설명

> 미션을 만드는 사용자(디자이너)가 ItemDetailView 에서 보는 **친근한 아이템 안내문 모음**.
> 각 아이템마다 "한 줄 효과 + 배치 팁" 한 쌍으로 구성. 빌더 picker 에 노출되는 16개 itemType 전부.

---

## 적용 위치 (예정)

- 화면: [`PlaySpot/Views/MissionBuilder/ItemDetailView.swift`](PlaySpot/Views/MissionBuilder/ItemDetailView.swift) 의 "아이템 정보" 섹션
- 제거: 기존의 `#코드`·`좌표` 표시
- 추가: 아래 각 아이템의 **한 줄 효과** + **배치 팁** 카드

```
┌─────────────────────────────────────────┐
│ [아이콘]  Quiz — 퀴즈                    │
│                                          │
│ 근처에 가면 퀴즈 창이 떠요. 정답을 맞혀야 │
│ 획득됩니다.                              │
│                                          │
│ 💡 여러 문제를 등록하면 매번 다른 게      │
│    출제돼요. 질문과 정답을 꼭 채우세요.   │
└─────────────────────────────────────────┘
```

각 itemType 별 텍스트는 [`PlaySpot/Models/ItemType.swift`](PlaySpot/Models/ItemType.swift) 에 `var detailGuide: (effect: String, tip: String)` 같은 헬퍼로 묶어 두는 걸 권장.

## 🚩 미션의 시작과 끝

### Start — 시작 깃발

| 항목 | 내용 |
|---|---|
| **효과** | 미션이 시작되는 출발점이에요. 플레이어가 이 위치를 찍어야 시간이 흐르기 시작합니다. |
| **배치 팁** | 미션에 꼭 1개. 안내문에 "여기서 출발하세요!" 같은 한 줄을 넣어주면 좋아요. |

### End — 결승 깃발

| 항목 | 내용 |
|---|---|
| **효과** | 미션이 끝나는 도착점이에요. 필수 아이템을 모두 모은 뒤 여기에 도달하면 미션 클리어! |
| **배치 팁** | 미션에 꼭 1개. 시작점에서 적당히 떨어진, 마지막에 들를 만한 위치에. |

## 🎯 미션의 핵심 요소

### Quiz — 퀴즈

| 항목 | 내용 |
|---|---|
| **효과** | 획득 하면 퀴즈 창이 떠요. 정답을 맞혀야 획득됩니다. |
| **배치 팁** | 여러 문제를 등록하면 플레이시시 랜덤으로 출제돼요.|

### Run Start — 타임어택 시작

| 항목 | 내용 |
|---|---|
| **효과** | 획득하는 순간 짧은 카운트다운이 시작돼요. 시간 안에 'Run End' 아이템을 획득 해야 합니다. |
| **배치 팁** | 'Run End' 아이템 자동으로 함께 생겨요. 제한 시간을 30~120초로 설정해 난이도를 조절하세요. |

### Run End — 타임어택 종료

| 항목 | 내용 |
|---|---|
| **효과** | Run Start 의 짝꿍이에요. 제한 시간 안에 획득하면 성공입니다. |
| **배치 팁** | 자동으로 생성되니 따로 추가할 필요 없어요. 위치만 옮기면 됩니다. |

## 💪 도움이 되는 보조 아이템

### Defense — 방어막

| 항목 | 내용 |
|---|---|
| **효과** | 획득하면 지뢰 한 번의 피해를 막아주는 방어 아이템이에요. |
| **배치 팁** | 안내문에 "지뢰 피해를 1번 막아드려요" 정도 넣어주세요. |

### Solution — 해답 부적

| 항목 | 내용 |
|---|---|
| **효과** | 획득하면 퀴즈 1개의 답을 알려줍니다.|
| **배치 팁** | 미니게임을 추가해서 획득 난이도를 조절하세요. |

### Stealth Radar — 스텔스 레이더

| 항목 | 내용 |
|---|---|
| **효과** | 획득 후 AR 화면에서 숨겨둔 아이템들이 보이게 돼요. |
| **배치 팁** | 미션에 1개만. AR 숨김 아이템과 함께 사용하세요. |

### Map Radar — 지도 레이더

| 항목 | 내용 |
|---|---|
| **효과** | 획득 후 지도 화면에서 숨겨둔 아이템들이 보이게 돼요. |
| **배치 팁** | 미션에 1개만, MAP 숨김 아이템과 함깨 사용하세요|

### Mine Radar — 지뢰 레이더

| 항목 | 내용 |
|---|---|
| **효과** | 획득 후 지도에 지뢰 위치와 폭발 반경이 표시돼요. |
| **배치 팁** | 미션에 1개만. 지뢰가 많은 코스라면 꼭 하나 두는 걸 추천! |

## ⚠️ 긴장감을 만드는 장애물

### Mine — 지뢰

| 항목 | 내용 |
|---|---|
| **효과** | 위험! 반경 안에 들어가면 폭발하면서 최근에 얻은 아이템 1개를 잃어요. |
| **배치 팁** | 반경(rangeAR)만 설정하면 끝. 좁은 길목·핵심 동선에 두면 긴장감이 살아납니다. 지도에 빨간 원으로 표시돼요. |

### Dark — 다크존

| 항목 | 내용 |
|---|---|
| **효과** | AR 보물찾기 존이에요. 지도 반경안에 아이템 표시가안되요 |
| **배치 팁** | 보물찾기 분위기를 만들고 싶은 구역에 활용하세요. |

## 🎁 재미와 보상

### Hint — 힌트

| 항목 | 내용 |
|---|---|
| **효과** | 획득 하면 힌트 문구가 보이는 안내 아이템이에요. |
| **배치 팁** | 미니게임을 추가 해서 난위도 조절 가능해요, 미션 완료에 대한 힌트나 안내말을 표시해요 |

### Gambling — 랜덤 박스

| 항목 | 내용 |
|---|---|
| **효과** | 획득하면 다른 아이템 중 하나가 랜덤으로 획득 할수 있어요. (행운 ✨) |
| **배치 팁** | 미션 곳곳에 흩뿌려두면 재미가 살아나요. 안내문 한 줄과 미니게임 선택 가능. |

### Coupon — 쿠폰

| 항목 | 내용 |
|---|---|
| **효과** | 획득하면 쿠폰 코드/안내문이 알림으로 떠요. |
| **배치 팁** | 'info' 칸에 실제 쿠폰 코드나 사용 안내를 적으세요. 예: `"이마트 5000원 할인 ABCD-1234"` / `"3월까지 사용 가능"`. |

## 💡 기타

### Store — 상점 *(준비 중)*

| 항목 | 내용 |
|---|---|
| **효과** | 미션 중 상점 진입점으로 쓸 예정인 아이템이에요. **지금은 효과가 없습니다.** |
| **배치 팁** | 향후 결제/포인트 화면 연동 예정. 현재는 배치해도 게임 중 알림이 뜨지 않으니 사용을 미뤄주세요. |

### 향후 작업 시 적용 단계

1. [`ItemType.swift`](PlaySpot/Models/ItemType.swift) 에 헬퍼 추가:
   ```swift
   extension ItemType {
       /// (게임 효과 한 줄, 디자이너 배치 팁) — 아이템 상세 화면 표시용.
       var detailGuide: (effect: String, tip: String) { ... }
   }
   ```
2. [`ItemDetailView.swift`](PlaySpot/Views/MissionBuilder/ItemDetailView.swift) 의 "아이템 정보" Section 교체:
   - **제거**: `#itemID · rawValue` 텍스트, 좌표 표시
   - **추가**: 아이콘 + 한글 이름 + `effect` 본문 + `💡 tip` 보조 박스
3. (선택) Store 아이템은 picker 에서 임시 숨김 처리 (`ItemPickerView.pickableTypes` 에서 제거).

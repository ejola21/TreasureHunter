# item_design2.md — 아이템 상세 화면 입력 필드

> 디자이너가 한눈에 이해할 수 있게 5개 입력 필드의 **이름**과 **한 줄 설명**만 정리.
> 적용 위치: [`PlaySpot/Views/MissionBuilder/ItemForms.swift`](PlaySpot/Views/MissionBuilder/ItemForms.swift)

---

## 5개 입력 필드

| 항목 이름 | 한 줄 설명 |
|---|---|
| **필수 여부** | 미션을 끝내려면 꼭 얻어야 하는 아이템인지 정해요. |
| **표시 방식** | 지도와 AR 화면에서 아이템이 어떻게 보일지 골라요. (숨김없음 / AR 숨김 / MAP 숨김) |
| **발견 거리** | 플레이어가 얼마나 가까이 가야 발견되는지 (단위 m). |
| **미니게임** | 아이템 획득 시 필요한 추가 동작 (없음 / 흔들기 게임 / 터치 게임 / 랜덤 게임). |
| **안내 문구** | 아이템을 얻은 순간 알림에 보여줄 문구를 적어요. |

---

## 미니게임 카테고리

| 값 | 라벨 | 설명 | 상태 |
|---|---|---|---|
| 0 | **없음** | 가까이 가면 바로 얻어요. | ✅ |
| 1 | **흔들기 게임** | 기기를 흔들어서 획득. | ✅ |
| 2 | **터치 게임** | 화면을 빠르게 터치해서 획득. | 🚧 준비 중 |
| 3 | **랜덤 게임** | 흔들기 / 터치 중 하나로 랜덤 배정. | 🚧 준비 중 |

---

## 화면 디자인 (안)

```
┌──────────────────────────────────────────────┐
│ ⓘ  Quiz — 퀴즈                                │
│    근처에 가면 퀴즈 창이 떠요…                  │
│ 💡 여러 문제를 등록하면 매번 다른 게 출제돼요.    │
├──────────────────────────────────────────────┤
│ 필수 여부                              [ ON ]  │
│   미션을 끝내려면 꼭 획득 해야해요.              │
├──────────────────────────────────────────────┤
│ 표시 방식                          [ 일반 ▾ ]  │
│   지도,AR화면에서 보일지 안보일지 설정해요.       │
├──────────────────────────────────────────────┤
│ 발견 거리                       [ − ] 30 m [+] │
│   AR화면에서 아이템이 표시되는 유효 반경          │
├──────────────────────────────────────────────┤
│ 미니게임                          [ 없음 ▾ ]   │
│   아이템 획득시 필요한 게임을 추가할 수 있어요.    │
├──────────────────────────────────────────────┤
│ 안내 문구                                      │
│ ┌──────────────────────────────────────────┐ │
│ │ "여기서 출발하세요!" 같은 한 줄을 적어요.  │ │
│ └──────────────────────────────────────────┘ │
└──────────────────────────────────────────────┘
```

- 각 필드 한 줄 아래에 **회색 작은 글씨로 1줄 설명** 표시
- 일부 아이템에서 의미 없는 필드는 자동으로 숨김

# 미션 설계서 — 화성행궁: 정조의 효심, 사도세자의 한

> [game_rule.md](game_rule.md) 의 디자인 패턴 D(탐험형) + B(Quiz 학습) 혼합.

| 항목 | 값 |

## 2. 스토리 아크 (4 Act)

### Act 1 — 왕의 도착 (Start)
정조가 어머니 혜경궁 홍씨를 모시고 도착한 신풍루. "이곳에서 잠시 머무르며, 아버지의 기억을 마주합니다."

### Act 2 — 사도세자의 그림자 (Dark + Stealth Radar 구간)
1762년 영조에 의해 뒤주에 갇혀 운명한 사도세자. 그 비극의 흔적은 표면에 드러나지 않습니다 — AR 로만 보이는 “기억 조각” 으로.

### Act 3 — 정조의 효심 (Quiz + Hint)
화성행궁 곳곳에 남은 정조의 효성. 어머니 회갑연(봉수당), 어진 봉안(화령전), 활쏘기로 마음을 다스린 득중정.

### Act 4 — 이루지 못한 꿈 + 클리어 (Run + End)
정조는 왕위를 물려주고 이곳에서 머물고자 했습니다. 그러나 갑작스러운 죽음으로 꿈은 멈췄습니다. 짧은 카운트다운 안에 봉수당에 도달하세요 — 마치 정조의 시간이 멈춘 그날처럼.

## 4. 퀴즈 변형 (아이템 #3, #8)

### 아이템 #3 — 중양문에서

| 질문 | 정답 후보 (변형) |
|---|---|
| 정조가 화성행궁을 지은 가장 큰 이유는? | **현륭원** (또는 "아버지 무덤") |
| 정조가 화성을 지으며 표방한 정치 이상은? | **개혁** (또는 "탕평") |
| 정조의 아버지 시호는? | **장조** |

→ 3개 변형 등록 → 매 플레이마다 랜덤 출제.

### 아이템 #8 — 봉수당에서

| 질문 | 정답 |
|---|---|
| 1795년 봉수당에서 열린 회갑연의 주인공은? | **혜경궁 홍씨** (또는 "혜경궁") |
| 정조가 어머니의 회갑을 기념한 행사 이름은? | **회갑연** (또는 "을묘원행") |
| 봉수당의 ‘봉수’ 가 뜻하는 것은? | **장수** |

→ 3개 변형 등록.

## 5. 디자인 패턴 — D + B 혼합 ([game_rule.md §4](game_rule.md))

```
Start(신풍루)
   ↓ Hint(좌익문 - 자유 진입)
   ↓ Quiz(중양문 - 첫 도전)
   ↓ Stealth Radar(어둠을 밝힘) ──┐
   ↓                          ↓
   Dark Zone(70m) 진입 ────────→ AR 숨김 Hint × 2 (사도세자 흔적)
   ↓
   ↓ Quiz(봉수당 - 회갑연)
   ↓ Hint(노래당 - 못 이룬 꿈)
   ↓ Solution(사관 - Quiz 안전망)
   ↓ Gambling(어보 - 깜짝 보상)
   ↓ Run Start(득중정) → 90s → Run End(봉수당) [클라이맥스]
   ↓
   End(화령전 - 어진 봉안)
```

핵심 디자인 의도:
- **다크존 + Stealth Radar 조합**: "사도세자의 흔적은 표면에 안 보인다" 의 시각적 표현
- **Run Start/End 90초**: 정조의 갑작스러운 죽음 = 시간이 멈추는 클라이맥스
- **Solution 안전망**: 역사 퀴즈로 좌절 방지
- **Gambling 어보**: 학문을 사랑한 왕의 캐릭터 연결

## 6. 빌더 입력 순서 가이드

1. **메타 입력** ([MissionSetupView](PlaySpot/Views/MissionBuilder/MissionSetupView.swift))
   - 제목/설명/언어/제한시간(45분)/가상모드 ON/뱃지 첨부
2. **지도 진입** → 화성행궁 줌인
3. **Start(#1) 배치** — 신풍루 longTap → ItemPicker → Start → 안내문 입력 → 저장
   - 자동으로 장소(Place) 가 "경기도 수원시..." 로 채워짐
4. **Dark(#4) 먼저 배치** — Dark Zone 반경 70m 시각 확인용 (다른 핀 배치할 기준)
5. **AR 숨김 Hint #5, #6 배치** — Dark 원 안에 두기
6. **Stealth Radar(#7) 배치** — Dark 입구 직전
7. **Hint(#2), Quiz(#3, #8) 배치**
8. **노래당(#9), Solution(#10), Gambling(#11) 배치**
9. **Run Start(#12) 배치** → 자동으로 Run End 페어 생성 → Run End 핀을 봉수당으로 드래그 → 제한 시간 90초 입력
10. **End(#13) 배치** — 화령전
11. **저장** — 비공개로 먼저 저장 → "테스트" 버튼으로 검증
12. **검증 통과 후 공개 토글 ON** → 다시 저장

## 7. 검증 체크리스트 ([game_rule.md §5](game_rule.md))

- [ ] Start, End 각 1개 (#1, #13)
- [ ] mandatory 아이템 ≥ 1 — 자동 mandatory 5개 (#1, #3, #7, #8, #12·Run End, #13) + 사용자 ON 가능
- [ ] Start ↔ End 거리 ≈ 150m (가깝지만 봉수당 경유로 코스 의미 충분)
- [ ] Quiz 변형 모두 등록 + 정답 입력
- [ ] Run Start ↔ Run End 거리 약 80m, 90초 = 도보 1.3m/s × 90s = 117m 여유 OK
- [ ] Mine 없음 → Mine Radar 안 둠 (올바름)
- [ ] Dark 반경 70m 안에 AR 숨김 Hint 2개 + Stealth Radar 진입 직전에 노출
- [ ] 안내문 11개 모두 한글로 채워짐 (이야기 흐름 통일)
- [ ] 뱃지 이미지 첨부

## 9. 비고

- 정조 24년(1800년) 정조의 갑작스러운 승하 = 미션 시간 제한과 Run Start/End 의 긴장감으로 정서적 연결
- 화령전(華寧殿) = "화성에서 평안하소서" — End 지점으로 의도적 선택 (정조의 안식)
- 다크존 70m = 봉수당 동측 잔디밭 정도. 실측 후 조정 권장
- 가상 모드 ON 으로 수원 외 지역 사용자도 체험 가능 — 관광 홍보 효과
